# 🤖 AI Chatbot Agent

A fully responsive AI chatbot application powered by API-based AI models.

---

# 🚀 Features

- Responsive modern UI
- Real-time AI chat
- API key integration
- Chat history
- Markdown support
- Typing animation
- Dark/Light mode
- Mobile friendly
- Authentication support
- Fast API responses

---

# 🛠 Tech Stack

## Frontend
- Next.js
- Tailwind CSS
- TypeScript

## Backend
- Node.js / Python API

## Database
- MongoDB

## AI Provider
- OpenAI API
- Gemini API
- Claude API
- Groq API
